# StayManager

StayManager is a React + TypeScript + Vite web application for managing apartment bookings, guests, and payments.

## Features

- View, create, update, and delete bookings
- Track apartment details and guest information
- Check-in and check-out functionality
- User authentication (signup, login, logout)
- Price summary and payment status
- Admin dashboard for bookings and apartments

## Tech Stack

- React 18 + TypeScript
- Vite
- TailwindCSS
- Supabase (Auth + Database)
- React Query for data fetching and state management
- React Router for routing
- date-fns for date formatting
- Recharts for displaying charts
